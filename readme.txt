# VAE（変分オートエンコーダ）実装

## プロジェクト概要
本プロジェクトはPyTorchを用いたVAE（変分オートエンコーダ）の実装です。猫と犬の画像データセットを使い、画像の生成・再構成・潜在空間の可視化を行います。

## ディレクトリ構成
- main.py: VAEの学習・テスト・画像保存
- model.py: VAEモデル本体
- plot_latent.py: 潜在空間の可視化（PCA）
- dataset/: 画像データセット（train/test, cats/dogs）
- result/: 学習済みモデルや生成画像の保存先

## 実装内容
- VAEモデル（エンコーダ・デコーダ・再パラメータ化トリック）
- 画像の前処理（リサイズ・正規化・データ拡張）
- 損失関数（再構成誤差＋KLダイバージェンス）
- 学習ループ（損失の記録・重み保存）
- テスト画像の再構成・保存
- 潜在空間のPCA可視化

## 使い方
1. dataset/ に画像を配置
2. main.py を実行して学習・再構成画像生成
3. plot_latent.py で潜在空間を可視化

## 必要なライブラリ
- torch
- torchvision
- matplotlib
- scikit-learn
